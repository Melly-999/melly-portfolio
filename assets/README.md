# Assets

This directory will hold production-ready assets for the Melly Portfolio
site. No generated assets are added as part of the repository bootstrap —
this file only documents the planned groups.

## Planned asset groups

- **Portfolio hero** — hero background / graphic for the home page
- **MellyCore posters** — static poster images for the MellyCore AIOS case study
- **MellyTrade Market Orbit** — static illustration/diagram for the MellyTrade case study
- **Screenshots** — sanitized, demo-data-only screenshots of MellyTrade and MellyCore AIOS
- **CV** — the approved CV PDF for download
- **Social preview image** — Open Graph / Twitter card image

## Rules

- No fake or misleading screenshots (no live-looking data, no invented metrics).
- Screenshots must be sanitized: no secrets, tokens, real account data, or personal information.
- Static/fixture data shown in any screenshot must be clearly labeled as such in the site copy.
